CORBA nodes sample

The CORBA nodes sample re-creates a scenario of managing stock in a product warehouse. The sample shows you how to:

  • Create a new stock item
  • Increase stock
  • View stock item details
  • Reduce stock
  • Delete the stock item

The stock control management system is hosted by a CORBA server Java application.

The sample uses the CORBARequest node to process the stock control requests, and action the responses.
